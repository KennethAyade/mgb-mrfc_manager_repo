# Assets

This folder contains branding and media assets for the MGB MRFC Manager project.

## Folder Structure

- **logos/** - Project logos and branding materials
  - Place your logo files here (PNG, SVG, etc.)
  - Can include multiple variants (light/dark, different sizes)

## Usage

These assets can be used for:
- Documentation
- README files
- Marketing materials
- Presentations
- Web interfaces
